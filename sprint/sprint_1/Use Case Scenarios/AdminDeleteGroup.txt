Context - User is logged in as an admin account

Primary Flow
1  - Admin navigates to View/Create/Edit/Delete Group Page on WebApp
2  - System navigates admin to view of all Groups with in application 
3  - Admin searches and selects Group to delete
4  - Admin selects delete Group button
5  - System reprompts Admin to ensure deletion of Group
6  - Admin verifies deletion of Group 
7  - System deletes Group 
8  - System notifies Admin of successful deletion of Group 
9  - Return to step 2

Alternate Flow(s)
ADMIN CANCELS DELETION OF GROUP
a1 - Starts at step 5
a2 - Admin selects cancel
a3 - Returns to Primary Flow step 2
